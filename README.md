# PolyTaxonomy
This data package supports the manuscript “A framework for identifying the polyploid complex in Rorippa (Brassicaceae): combining trait evolution, herbarium records, and machine learning.” https://doi.org/10.1093/aob/mcag050. 

It contains all data and R scripts used in trait evolution modeling, kernel density analysis, species distribution modeling (SDM), and decision tree classification. The dataset includes spatiotemporally referenced occurrence records from field (2017–2020) and herbarium (1893–2021) collections. Raw input data include morphological traits (e.g., seed arrangement, petal number, leaf and fruit morphology), ploidy level estimates, phenological indexing, phylogeny, and georeferenced specimen metadata. Cleaned and thinned datasets for SDM analysis and formatted binary classification datasets for machine learning are also provided. Accompanying R scripts allow full replication of all spatial, ecological, and classification analyses presented in the study. These materials are intended to support reproducibility and facilitate further research on polyploid species delimitation, trait evolution, and biodiversity modeling.
